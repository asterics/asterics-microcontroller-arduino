AsTeRICS Arduino Microcontoller Demo

Objective

 Digital Input/Output, PWM and ADC with the microcontroller Arduino Uno.

Description

 The Arduino Uno microcontroller platform supports digital input and output and analog input. This tutorial shows those functions in four demonstrations:
  • Digital Output
  • PWM
  • ADC
  • Servo PWM

Requirements

  • USB Port
  • USB cabel (USB 2.0 - A/B)
  • Arduino Uno SMD R3
  • AsTeRICS 3.0 installed and ARE running
  • OS: Windows, Linux (incl. RPi), Mac OS X

Additionally required for demos:

  • Digital Output: LEDs, Resistors
  • PWM: LEDs, Resitors
  • ADC: Capacitor, Resistor
  • Servo PWM: Servo motor

Setup CIM w. Arduino

Before Arduino can be used with WebACS, the

  • Windows 7, 8, 10

    • Flash Arduino Uno 
     C:\git\asterics\AsTeRICS\CIMs\Ardunio\build> flash

  • Linux (Debian, Ubuntu, ...)

    • Install avrdude 
     $ sudo apt-get install avrdude
    Flash Arduino Uno 
     $ avrdude -pm328p         \ 
               -P /dev/ttyACM1 \ 
               -c stk500v1     \ 
               -b 115200       \ 
               -I flash:w:~/git/asterics/AsTeRICS/CIMs/Ardunio/build/Arduino.hex \ 
               -F

  • Mac OS X

    • Install avrdude 
     $ brew install avrdude --with-usb
    Flash Arduino Uno 
     $ avrdude -pm328p         \ 
               -P /dev/ttyACM1 \ 
               -c stk500v1     \ 
               -b 115200       \ 
               -I flash:w:~/git/asterics/AsTeRICS/CIMs/Ardunio/build/Arduino.hex \ 
               -F
