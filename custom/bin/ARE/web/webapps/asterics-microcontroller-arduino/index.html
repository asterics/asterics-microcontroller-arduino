Arduino Demo

This is a tutorial including four different demos, showing you how to use the popular arduino microcontroller platform together with AsTerICS.

Learn more Â»

Objective

Digital Input/Output, PWM and ADC with the microcontroller Arduino Uno.(current)

Description

The Arduino Uno microcontroller platform supports digital input and output and analog input. This tutorial shows those functions in four demonstrations:

  • Demo 1: Digital Output
  • Demo 2: PWM
  • Demo 3: ADC
  • Demo 4: Servo PWM

Requirements

For this demo you need

  • USB Port
  • USB cabel (USB 2.0 - A/B)
  • Arduino Uno SMD R3 with AsTeRICS CIM installed
  • AsTeRICS 3.0 installed and ARE running
  • OS: Windows, Linux (incl. RPi), Mac OS X

Additionally required for demos:

  • Demo 1: Digital Output: LEDs, Resistors
  • Demo 2: PWM: LEDs, Resitors
  • Demo 3: ADC: Capacitor, Resistor
  • Demo 4: Servo PWM: Servo motor

Install CIM on Arduino Uno

Windows 7, 8, 10

Flash Arduino Uno

 C:\git\asterics\AsTeRICS\CIMs\Arduino\build> flash

Linux

Install avrdude

$ sudo apt-get install avrdude

Flash Arduino Uno

 $ avrdude -pm328p         \ 
           -P /dev/ttyACM1 \ 
           -c stk500v1     \ 
           -b 115200       \ 
           -I flash:w:~/git/asterics/AsTeRICS/CIMs/Arduino/build/Arduino.hex \ 
           -F

MAC OS X

Install avrdude

 $ brew install avrdude --with-usb/

Flash Arduino Uno

 $ avrdude -pm328p         \ 
           -P /dev/ttyACM1 \ 
           -c stk500v1     \ 
           -b 115200       \ 
           -I flash:w:~/git/asterics/AsTeRICS/CIMs/Arduino/build/Arduino.hex \ 
           -F

Demos

Demo 1: Digital Output

The first demo introduces the digital output capabilities that are included with the Arduino Uno platform. Setup the circuit and press the buttons in the application section.

Circuit

For this demo you have to setup an electric circuitry as depicted below.

First, connect a 220 Ω resistor (R1) to the digital output pin with the number 2 on the right side of the connectors of the Arduino Uno platform, as shown in the left picture above. Then connect the anode of the LED (D1 - positive) with that resistor. Finally, connect the cathode of the LED (negative) to one of the GND pins on the left side.
